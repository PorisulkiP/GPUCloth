Application Icons (bpy.app.icons)¶

bpy.app.icons.new_triangles(range, coords, colors)¶
Create a new icon from triangle geometry.
Parameters

  • range (tuple.) – Pair of ints.

  • coords (byte sequence.) – Sequence of bytes (6 floats for one triangle) for (X, Y) coordinates.

  • colors (byte sequence.) – Sequence of ints (12 for one triangles) for RGBA.

Returns

Unique icon value (pass to interface icon_value argument).

Return type

int

bpy.app.icons.new_triangles_from_file(filename)¶
Create a new icon from triangle geometry.
Parameters

filename (string.) – File path.

Returns

Unique icon value (pass to interface icon_value argument).

Return type

int

Release the icon.
